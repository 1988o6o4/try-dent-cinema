# トライデントシネマ

  

## サイト概要
 - 映画やその他映像の依頼や告知が可能なサイト
  



  

### サイトテーマ
- 映像作成などの仕事の依頼ができる
- 舞台などの告知ができる
  



  

### テーマを選んだ理由

 - 知人の会社の紹介

  

### ターゲットユーザ

  

- 映画などの作成または、舞台などの仕事をしている方
- 舞台などを見に行きたい方

  

### 主な利用シーン

  

- 仕事の依頼や告知する時
